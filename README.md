# Praca inzynierska
 Przegląd wzorców projektowych wykorzystywanych przy programowaniu gier w silniku Unity

Celem jest zaprojektowanie oraz implementacja aplikacji w silniku Unity, która posłuży jako interaktywny przewodnik po wybranych wzorcach projektowych stosowanych w programowaniu gier. W ramach projektu stworzona zostanie gra składająca się z kilku niezależnych mini gier, z których każda będzie prezentowała zastosowanie innego wzorca projektowego, takich jak m.in. Singleton, Factory, Object Pool, MVC czy Observer. Dzięki temu gra będzie pełniła funkcję edukacyjną, pozwalając graczowi nie tylko poznać konkretne wzorce, ale również zrozumieć kontekst ich użycia i sposób implementacji w praktycznym środowisku.

W części teoretycznej pracy zostaną szczegółowo przeanalizowane wybrane wzorce projektowe, wraz z ich charakterystyką, typowymi zastosowaniami oraz wpływem na strukturę i organizację kodu. Każdy wzorzec zostanie przedstawiony z uwzględnieniem jego mocnych i słabych stron, potencjalnych kierunków rozbudowy, a także uzasadnienia jego użycia w określonych kontekstach projektowych.
